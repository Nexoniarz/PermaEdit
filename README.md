# PermaEdit

Make permanent visual edits to any webpage.

PermaEdit is a lightweight developer tool (WebExtension) designed to freeze and preserve manual HTML modifications. Unlike standard browser Developer Tools (F12) which wipe your changes upon refreshing the page, PermaEdit hooks into the browser storage to ensure your visual tweaks stay intact even after a page reload ($F5$).

## Features

* **F12 State Persistence:** Modify any webpage using your browser's native Developer Tools, then lock those changes permanently with a single click.
* **Quick-Save Context Menu:** Right-click anywhere on the webpage to instantly save your current F12 modifications without opening the extension popup.
* **Native Theme Integration:** Clean and minimal user interface that automatically adapts to your system's Light or Dark mode.
* **One-Click Reset:** Easily wipe cached modifications and restore the original webpage back to default instantly.
